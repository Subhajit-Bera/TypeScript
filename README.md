

## Class & Object In TypeScript
```
class Class_name {
	private height;
	public weight;
	
  constructor( height:number,weight:number){
	  this.height=height;
	  this.weight=weight
  }
}

const obj1=new  Class_name(height,weight);
console.log(obj1.weight);

```
